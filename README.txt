Input
----------------------------------
The application requires the user to set the following things in Utils.py file.

1. PATH - The path of the input and output folders. The input folder contains the input file/files/ The output folder will eventually contain the output file after executing the program.
2. filename - The name of the input file to be fed to the application.
3. keep_cols - The names of the columns which are to be kept by the filtering application.
4. street_column - Name of the street name column.
5. house_number_column - Name of the house number column.
6. postcodes_column - Name of the postcode column.

Execution
----------------------------------
After all the input parameters are set in the Utils.py file, run the main.py file using

$ python main.py
